게임 컨셉 - 노래 타이밍에 맞춰서 버튼을 누르는 형식의 리듬 게임

개발 범위 -  버튼을 누르면 판정에 따라 Fail, Bad, Good, Great, Perfect로 나뉘어 점수를 얻는다.
         -  음악을 선택하고 음악에 따라 버튼이 나오는 형식이 다르다.
         -  설정에서 버튼이 나오는 속도같은 형태를 바꾼다.
         -  게임의 퍼펙트 정도에 따른 랭크를 부여한다.
